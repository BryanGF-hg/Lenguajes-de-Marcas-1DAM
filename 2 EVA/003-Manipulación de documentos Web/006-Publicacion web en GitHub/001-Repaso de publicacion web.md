Que hace falta para publicar una web?

Dominio - loquesea.dominio
loquesea = nombre del dominio
.dominio = extensión

La combinación entre ambos elementos debe estar libre (Nadie antes de haber registrado esa combinación)

Ejemplo:
en el caso del profesor: jocarsa = nombre del dominio
en el caso del profesor: .com = Extensión que quiere decir

Los dominios son mnemonicos (evitan que tengamos que recordar las IP de los servidores)

En los años 80 extensiones originales de dominio:

.net
.org
.biz

Cuanto cuesta un dominio?
Depende del dominio

Dominios genéricos (.com, net, .es) - coste aprox de 15 euros al año

Los dominios comerciales pueden tener un coste mayor
30-100 euros al año

El dominio se tiene que renovar
Si no se renueva, se pierde
